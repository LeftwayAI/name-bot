## Conversion Tracking Setup

### Step 1: Create Google Ads Conversion Action

1. Go to Google Ads â†’ **Tools & Settings** â†’ **Conversions**
2. Click **+ New Conversion Action**
3. Select **Website**
4. Enter conversion details:
   - **Category:** Purchase
   - **Value:** $49 (use the same value for each conversion)
   - **Count:** One (only count unique purchases)
   - **Attribution model:** Last click
5. Click **Create and Continue**
6. **Tag Setup:** Use Google Tag Manager OR install Global Site Tag

### Step 2: Install Conversion Tracking Code

Add this to `/app/success/page.tsx` (the page users see after payment):

```typescript
// Add this useEffect to trigger conversion tracking
useEffect(() => {
  // Google Ads Conversion Tracking
  if (typeof window !== 'undefined' && (window as any).gtag) {
    (window as any).gtag('event', 'conversion', {
      'send_to': 'AW-XXXXXXXXX/XXXXXXX', // Replace with your conversion ID
      'value': 49.0,
      'currency': 'USD',
      'transaction_id': searchParams.get('name') || ''
    });
  }

  // Facebook Pixel (optional, if running Meta ads)
  if (typeof window !== 'undefined' && (window as any).fbq) {
    (window as any).fbq('track', 'Purchase', {
      value: 49.0,
      currency: 'USD'
    });
  }
}, [searchParams]);
```

### Step 3: Install Global Site Tag

Add to `/app/layout.tsx` in the `<head>`:

```typescript
<Script
  async
  src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"
  strategy="afterInteractive"
/>
<Script id="google-ads" strategy="afterInteractive">
  {`
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'AW-XXXXXXXXX');
  `}
</Script>
```

Replace `AW-XXXXXXXXX` with your Google Ads conversion ID.

## Daily Monitoring Checklist

### First 3 Days (Critical Testing Phase)
Check TWICE daily (morning + evening):

1. **Impressions:** Are ads showing? (If <10/day, increase bids)
2. **Clicks:** Getting clicks? (If <5/day, improve ad copy or increase bids)
3. **CTR:** >2%? (If <1%, ad copy needs work)
4. **CPC:** Within budget? (If >$2, pause expensive keywords)
5. **Conversions:** Any sales? (If 0 after 50 clicks, debug funnel)
6. **Cost per Conversion:** <$100? (If >$150, pause and optimize)

### Days 4-10 (Scaling Phase)
Check ONCE daily:

1. **Identify winners:** Which campaigns/keywords convert best?
2. **Reallocate budget:** Move money from low-performers to winners
3. **Pause losers:** Keywords with >30 clicks + 0 conversions
4. **Add negative keywords:** Irrelevant searches wasting budget
5. **Test ad variations:** A/B test headlines and descriptions
